Project Overview

The Dawlance Cost Reduction Tracker is a comprehensive web application designed to help Dawlance manage and track cost reduction initiatives across different factories. The system features a multi-role architecture with specific functionalities for each user type, enabling efficient project management, ROI analysis, and progress tracking.

Multi-Role System

Admin, Manager, and User roles with specific permissions and workflows

Advanced Analytics

ROI calculations, milestone tracking, and comprehensive reporting

Multi-Factory Support

Manage projects across multiple Dawlance manufacturing facilities

Admin Functionalities

Complete system administration and oversight capabilities

Dashboard

  • Overview of total users, projects (total, pending, approved)
  • Recent projects display with status indicators
  • Quick access to key administrative functions
  • Access path: /admin/dashboard

Projects Management

  • View, edit, and delete all projects
  • Filter projects by year and month
  • Bulk operations and project status management
  • Access path: /admin/projects

Data Upload

  • Upload Excel/CSV files containing factory data
  • View upload history and validation results
  • Data integrity checks and error reporting
  • Access path: /admin/upload

Unit Costs Management

  • Manage and update unit costs for different factories
  • Parameter cost configuration and validation
  • Historical cost tracking and analysis
  • Access path: /admin/unit_costs

Account Management

  • Add, edit, enable/disable, and delete user accounts
  • Manage all roles (Admin, Manager, User)
  • Role-based permission management
  • Access path: /admin/manage_accounts/<role_type>

Project Categories

  • Add, edit, and delete project categories
  • Manage Kaizen, Six Sigma, and other methodologies
  • Category-specific workflow configuration
  • Access path: /admin/project_categories

Manager Functionalities

Project oversight, approval workflows, and performance analytics

Dashboard

  • Overview of total investment, monthly savings, total projects
  • Pending reviews and recent projects display
  • Key performance indicators and metrics
  • Access path: /manager/dashboard

Analytics Dashboard

  • Detailed charts for monthly savings trend
  • Project type distribution visualization
  • Cumulative savings analysis using ApexCharts
  • Access path: /manager/analytics_dashboard

Projects Management

  • View all projects with filtering capabilities
  • Filter projects by type, year, and month
  • Export all projects to Excel with formatting
  • Access path: /manager/projects

Project Review & Approval

  • View detailed project information
  • Update project status (Pending, Approved, Rejected)
  • Add comments and feedback to projects
  • Access path: /manager/project_detail/<project_id>

Technical Specifications

  • Tailwind CSS for styling
  • JavaScript ES6+ with modern features
  • Chart.js for data visualization
  • Font Awesome icons

Backend Technologies

  • Flask web framework
  • SQLAlchemy ORM
  • Pandas for data processing
  • XlsxWriter for Excel exports
  • Session-based authentication
